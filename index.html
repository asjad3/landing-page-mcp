v1.0/release

Creative tools for your agent

An MCP server for ImagineArt. Eight creative endpoints (image, video, music, scripts, upscaling, and more) for Claude, Cursor, and any client that speaks the Model Context Protocol.

01

Open your client

Launch Claude Desktop. Open Settings, Connectors, Add custom connector.

02

Add the server

Name it Imagine. Paste this command:

npx -y @imagine/mcp-server
03

Connect and create

Click Add, Connect. Sign in with your imagine.art account. Ask Claude to generate an image.

Using Claude Code or Codex? The CLI works best.

03/toolset

Image. Video. Music. Scripts. Plus four more.

Eight endpoints, every one an MCP tool call from your agent. Your imagine.art account, your model selection, your asset library.

Text to Image

ImagineArt 1.5. Hyper-real lighting, accurate text, around 4 seconds per render. Full prompt control, model selection, and aspect ratios.

imagine.text_to_image()

Text to Video

5 to 10 second clips. Seedance 2.0.
